PATIENT-WISE EZ CLASSIFICATION
HYPERPARAMETERS AND EVALUATION SETTINGS

Shared settings
---------------
Features:
  - Hq Value
  - evec avg
  - deltaH
  - frac_Avg

Target:
  is_soz: 0 = Non-EZ, 1 = EZ

Classification threshold:
  0.50

Outer validation:
  10 patient-wise folds

Adult split method:
  Predefined manual patient folds

Pediatric split method:
  StratifiedGroupKFold(
      n_splits=10,
      shuffle=True,
      random_state=42
  )

Missing-value handling:
  SimpleImputer(strategy="median")
  Fitted on the training fold only

SMOTE:
  sampling_strategy="auto"
  k_neighbors=min(5, minority_training_count - 1)
  random_state=42 + fold_number
  Applied to the training fold only
  Test data are never oversampled

Random-state policy:
  Base random state = 42
  Model and SMOTE random state = 42 + fold_number

Output figure:
  PNG, 600 DPI


1. Adult Logistic Regression
----------------------------
Preprocessing:
  SimpleImputer(strategy="median")
  StandardScaler()
  SMOTE(training fold only)

Model:
  LogisticRegression(
      C=np.inf,
      solver="lbfgs",
      max_iter=10000,
      random_state=42 + fold_number
  )

C=np.inf makes the L2 penalty contribution effectively zero, providing an
unregularized logistic-regression fit while using the lbfgs solver.

Hyperparameter-selection note
-----------------------------
These scripts use fixed hyperparameters. No hyperparameter search or tuning is
performed on the outer test folds. Any future tuning should be performed only
inside the training patients of each outer fold using nested group-aware cross-validation.
